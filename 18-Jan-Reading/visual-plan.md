- **Row 3**: **WHERE** -> Places (Icon: Map Pin).

### Slide 9.1: Task 1 - Answer 1
- **Answer**: "...**which / that** are grown in this country..." (Defining)
- **Snippet**: "Common foods [which/that are grown in this country] include..."
- **Explanation**: Defines *which* foods. 'That' is okay.

### Slide 9.2: Task 1 - Answer 2
- **Answer**: "...**who / that** sell in the local market..." (Defining)
- **Snippet**: "Farmers [who/that sell in the local market] always sell..."
- **Explanation**: Defines *which* farmers. 'That' is okay.

### Slide 9.3: Task 1 - Answer 3
- **Answer**: ", **which is an English dessert**," (Non-Defining)

### Slide 11.2: Task 2 - Answer 2

### Slide 11.4: Task 2 - Answer 4
